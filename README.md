
# Secuton Task Scheduler

Secuton is a task-scheduler application which allows users to schedule their daily (routine) tasks, set-up reminders and get customized notifications for completing TODOs with a deadline, and takes into consideration the mental-health of users and also has a section for scheduling leisure tasks!  



![iR2tADVGYuBo_1024_500](https://user-images.githubusercontent.com/75166805/208032363-b7db7fd7-075a-41a9-abbd-14f32e77c638.png)




## Features

- Verified User Login - Each account is verified through e-mail verification before creating an account.
- Routine Tasks - Schedule your tasks based on pre-defined slots and day.
- TODO Tasks - Create a TODO list with deadlines and get customized notifications and alarms for managing your TODOs.
- Leisure Activities - Apart from one's ambition, the app also lets users create and maintain a Leisure Activity list to help the user take a chill-pill!
- Access from anywhere!

## Features (Tech)
- Firebase onEventListener Socket API used for Realtime upadates.
- Clean Architecture (on MVVM) used.
- Dagger-Hilt Dependency Injection used.
- Airbnb Epoxy Library used for EpoxyRecyclerView implementation.
